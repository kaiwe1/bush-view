# BushView

英雄联盟召唤师查询与对局数据分析工具

## 功能

- 召唤师查询
- 个人资料

## 路线图

- [ ] 对局即时详情： 自动检测当前对局，实时展示队友常用英雄、胜率及近期状态。
- [ ] 自动接受对局、自动禁用英雄
- [ ] 版本英雄榜单
